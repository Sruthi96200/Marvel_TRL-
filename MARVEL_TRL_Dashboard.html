Architecture

Subsystem → components · headings link to TRL cards

Heat Transport System (Heat Pipes)

  • Na/NaK heat pipe units (evaporator, adiabatic section, condenser)
  • Wick architecture and capillary return performance under transients
  • Heat-pipe manifold / array layout for core-to-convertor coupling
  • Envelope materials and seals at sustained temperature and chemistry
  • SPHERE/PCAT-style long-duration and startup/shutdown test context

Reactor Core and Fuel System

  • Fuel element geometry and enrichment assumptions (TRIGA/SNAP heritage context)
  • Yttrium hydride moderator behavior (hydrogen retention, temperature effects)
  • Primary core structural internals and dimensional tolerances
  • NaK thermal-hydraulic field in channels/plenum and hot-spot control
  • Coupled neutronics + thermo-mechanics model validation against experiment

Power Conversion System (Stirling Engines)

  • Stirling/free-piston convertor thermal interface to condenser side
  • Alternator/power electronics path to conditioned electric output
  • Load-following and dynamic control under reactor transients
  • Reliability and maintenance envelope (seals, bearings, vibration)
  • Heritage transfer from space systems to nuclear-site qualification context

Control and Instrumentation Systems

  • Microreactor Automated Control System (MACS) supervisory control logic
  • Sensor suite (temperature, flow, neutron/proxy) and validation chain
  • Actuation path for startup/shutdown/reactivity management
  • Remote monitoring, cyber-hardening, and fault handling behavior
  • Digital I&C V&V evidence required for licensing acceptance

Safety and Shutdown Systems

  • Passive decay heat removal architecture and heat-sink assumptions
  • Shutdown / scram logic and fail-safe actuation path
  • Reactivity feedback mechanisms and off-normal response envelope
  • Accident/transient analysis set (startup, load rejection, cooldown events)
  • Licensing-basis artifacts (safety case, acceptance criteria, margins)

Grid Integration and Load Coupling

  • Microgrid / data-center load coupling use cases and duty cycles
  • Interconnect and power-quality path (switchgear, inverter/protection)
  • Dispatch/load-following control coordination with plant controls
  • Islanded operation, black-start/restart, and transition behavior
  • End-user demonstration scope vs. reactor demonstration scope
